Using h:dataTable in JSF 2


The following examples illustrate some of the h:dataTable options that are explained in the tutorial.

  • Basics. The var and value attributes of h:dataTable, and the nested h:column element.
  • Headers and captions. f:facet with name = "header", "caption", "footer", etc.
  • CSS styles. styleClass, captionClass, headerClass, footerClass, rowClasses, columnClasses.
  • Ajax. Updating a table without reloading the entire page.
  • Conditional output and editable cells. Using the "rendered" attribute to conditionally output h:inputText or h:outputText, thus making table cells editable by the user.
