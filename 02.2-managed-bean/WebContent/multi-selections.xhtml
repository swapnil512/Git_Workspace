Selecting Elements from Choices

Using h:selectManyblah instead of h:selectOneblah. Notice that h:selectManyMenu is rendered very differently from browser to browser, and is quite hard to use in Firefox. Skip h:selectManyMenu and use h:selectManyListbox or h:selectManyCheckbox instead. In all three cases, you use an array or List of selected values instead of a single one.

h:selectManyMenu

h:selectManyListbox

h:selectManyCheckbox
